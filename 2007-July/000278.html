Hello.

I'm checking the UPGRADE.txt [1] file to see how much work we need to do 
before fully supporting Asterisk 1.4, and I came to this initial list:


*Change ${TIMESTAMP} variable to ${STRFTIME(${EPOCH},,%Y%mNaVH%M%S)}

*AgentCallBacklogin is depricated -> See [2]

*pbx_functions are now splited in several modules.

*n+101 return code is depricated.

*Use rfc2833compensate option in sip.conf to comunicate with 1.2 
asterisk systems.

*Set CALLERID(ANI) as well as CALLERID(num).

*Check musicclass and  mohsuggest stuff.
